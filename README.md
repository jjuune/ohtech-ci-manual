# 02. Design Principles

브랜드를 표현하는 디자인의 기본 원칙입니다.

### 2.1 Scientific Minimalism
- 장식보다 정보 전달을 우선
- 과학적 신뢰감과 절제된 레이아웃 유지

### 2.2 Premium Industrial
- 하이엔드 산업 디자인 감성

### 2.3 Human First
- 사용자 중심의 직관적인 경험

### 2.4 Quiet Confidence
- 과장 없이 신뢰감을 전달하는 표현

### 2.5 Invisible Technology
- 기술을 과시하지 않고 자연스럽게 전달

### 2.6 Precision over Decoration
- 장식보다 정확성과 기능성을 우선
